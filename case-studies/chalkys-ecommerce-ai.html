Problem

  • Chalkys operated a Shopify-based vinyl record marketplace with 500,000+ products, but product descriptions were inconsistent, incomplete, and manually written — costing significant labour hours and limiting search discoverability.
  • Missing metadata (tracklists, artist info, condition details) meant lower conversion rates and increased customer support enquiries.
  • Manual data entry processes couldn't scale with the growing catalogue, and existing ETL workflows were fragile and slow.

Solution

I engineered an AI-powered product automation pipeline integrating OpenAI for description generation, Google Search API and Discogs API for data enrichment, and custom ML models for metadata extraction and verification. The system was deployed on AWS with automated ETL pipelines that processed the entire catalogue continuously, replacing manual workflows with intelligent automation.

Key engineering highlights

  • Integrated OpenAI API to generate SEO-optimised product descriptions at scale, producing consistent, high-quality content for 500,000+ products — improving organic search visibility by an estimated 40%.
  • Built automated data enrichment pipelines using Google Search API and Discogs API to fill missing product metadata (tracklists, artist information, release dates), achieving 95%+ data accuracy across the catalogue.
  • Designed machine learning models to extract, verify, and format critical product data from unstructured sources — reducing manual data entry workload by 70% and saving an estimated £85K+ annually in labour costs.
  • Architected AWS-based ETL pipelines using Lambda, S3, and SQS for fault-tolerant batch processing, reducing catalogue update time from hours to minutes with automatic retry and dead-letter handling.
  • Implemented intelligent product categorisation using NLP-based classification, enabling customers to discover products through genre, era, and condition filters — contributing to a measurable uplift in conversion rates.
