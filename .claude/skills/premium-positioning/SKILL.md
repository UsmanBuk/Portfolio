# Premium Positioning (£1,600-£2,000/day)

## Identity

- **Primary**: AI Systems Architect | Healthcare Technology
- **Secondary**: Enterprise RAG & LLM Specialist
- **Avoid**: "Full Stack Engineer", "Web Developer"

## Differentiation

> "The only engineer who has delivered production RAG systems across NHS with Azure AI, handling 3,000+ users with 94% success rate and zero security incidents."

## Content Rules

### Always Include
- Team size led/mentored
- Budget owned/influenced
- C-level stakeholder engagement
- Quantified outcomes with £ values

### Writing Style
- First person: "I architected..." not "The system was architected..."
- Action verbs: Architected, Designed, Led, Delivered
- Specific numbers over vague claims

### Avoid
- Weak verbs: helped, assisted, supported
- Vague quantities: various, many, multiple
- Generic titles without AI/ML qualifier
