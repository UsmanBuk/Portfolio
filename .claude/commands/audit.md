# Portfolio Audit

Perform a comprehensive audit of the portfolio website. Check for:

## Content Review
- Outdated information (roles, dates, project descriptions)
- Broken or placeholder links
- Missing alt text on images
- Spelling and grammar issues

## Technical Health
- Broken internal links between pages
- Missing or incorrect meta tags (title, description, og tags)
- Images without lazy loading attribute
- Console errors or JavaScript issues
- CSS issues (unused styles, specificity problems)

## Performance
- Large unoptimized images
- Render-blocking resources
- Missing preload/prefetch hints for critical assets

## Accessibility
- Color contrast issues
- Missing ARIA labels where needed
- Keyboard navigation problems
- Screen reader compatibility

## SEO
- Missing structured data
- Duplicate or missing page titles
- Missing canonical URLs

## Output
Provide a prioritized list of issues found, grouped by severity:
1. **Critical** - Broken functionality or major content errors
2. **Important** - Accessibility or SEO issues
3. **Minor** - Nice-to-have improvements

For each issue, specify the file and line number where applicable.
